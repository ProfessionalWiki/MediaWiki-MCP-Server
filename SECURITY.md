# Security Policy

- Use [GitHub](https://github.com/ProfessionalWiki/MediaWiki-MCP-Server/security/advisories/new) to report a new vulnerability.
- All communications and disclosures are handled within the [GitHub Security Advisory](https://github.com/ProfessionalWiki/MediaWiki-MCP-Server/security/advisories). 
- We will respond within 5 working days.
